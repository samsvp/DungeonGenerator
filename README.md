# DungeonGenerator

A simple turn-based RPG dungeon generator.

## Types of Room
- Combat rooms;
- Healing rooms;
- Vendor rooms;
- Breather rooms;
- Boss room;
- Initial player spawn room;

The dungeons have a FOE system support, that is, stronger elite enemies that the player can choose to avoid or risk battle for rewards.
